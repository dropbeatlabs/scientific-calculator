# Scientific Calculator

## Project Overview
This project is a comprehensive scientific calculator designed to facilitate a wide range of mathematical operations. It supports:

### Feature Set

1. **Arithmetic Operations**:
   - Addition (+)
   - Subtraction (-)
   - Multiplication (×)
   - Division (÷)

2. **Exponential Functions**:
   - Raises numbers to a power (e.g., x^n)
   - Supports fractional and negative exponents

3. **Root Calculations**:
   - Square roots (√)
   - Higher-order roots (e.g., 3√)

4. **Trigonometric Functions**:
   - Sine (sin)
   - Cosine (cos)
   - Tangent (tan)
   - Inverse functions (asin, acos, atan)

5. **Hyperbolic Functions**:
   - Hyperbolic sine (sinh)
   - Hyperbolic cosine (cosh)
   - Hyperbolic tangent (tanh)

6. **Inverse Functions**:
   - Allows calculation of inverse trigonometric and hyperbolic functions.

7. **Logarithmic Operations**:
   - Base 10 logarithm (log)
   - Natural logarithm (ln)
   - Logarithm with configurable base

8. **Scientific and Engineering Notation**:
   - Support for input and output in scientific (e.g., 2.5e3) and engineering notation.

### Usage
The calculator can process inputs written in standard mathematical notation, and it will output results in a user-friendly format.

### Conclusion
This scientific calculator is suitable for students, engineers, and anyone needing advanced mathematical computations.
